## 📁 Project Access
To try it is very simple:<br>
-Clone the repository
```bash
$ git clone https://github.com/julianofrodrigues/My-Skills-App
```
-Install
```bash
$ yarn install
```
or
```bash
$ npm install
```

-Start the App
```bash
$ npm start
```
or
```bash
$ yarn start
```
Connect your phone or set up an emulator to preview the app.
